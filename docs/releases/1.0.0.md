# 1.0.0

Initial public release of Yonsei Dev Index.

## Highlights

- GitHub OAuth registration and sign-in
- Public people directory with search, sorting, and verification filter
- Profile management for optional name and major fields
- Yonsei email verification with a visible verified marker
- Production deployment on Vercel, Render, Supabase, Resend, and Cloudflare

## Notes

- This release established the first live MVP at [ysdevidx.com](https://ysdevidx.com).
- The product scope intentionally stayed narrow: a GitHub-based people index, not a community platform.
