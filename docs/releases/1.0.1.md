# 1.0.1

Post-release hardening and product expansion update.

## Summary

`1.0.1` is the first real patch release after the initial public launch.
It tightened the verification flow, improved loading behavior under free-tier hosting constraints, and introduced the first version of the Organizations workflow.

## Product Changes

- Added the public `Organizations` page
- Added organization search, sorting, and pagination
- Added organization kinds such as `Campus Org`
- Added organization request submission flow for logged-in users
- Added admin review flow for organization requests
- Added applicant-side request status view in `My Profile`
- Added optional decline notes for rejected organization requests

## UX Improvements

- Reduced login-state button flicker on the public pages
- Moved public directory loading to the client to avoid Vercel timeouts while Render wakes from sleep
- Added softer loading states instead of hard failure messaging during backend wake-up
- Continued cleanup of profile, verification, and organization management screens
- Added clearer status badges for organization request states

## Security Hardening

- Removed production `localhost` credentialed CORS allowance
- Added same-origin checks for authenticated state-changing requests
- Added browser security headers on the web app
- Hardened verification request throttling
- Generalized verification error responses to avoid leaking too much state
- Enforced one verified Yonsei email per account

## Operations Notes

- This release assumes the web app may render before the free-tier backend is awake
- Organization data remains intentionally curated and small
- Admin review remains explicit instead of auto-publishing submissions

## Migration Notes

- `20260320_000004` adds `organizations`
- `20260322_000005` adds `organization_submissions`
- `20260322_000006` adds `review_note` to organization submissions

## Result

`1.0.1` turns the initial launch into a safer and more operationally stable baseline, while keeping the product scope intentionally narrow.
